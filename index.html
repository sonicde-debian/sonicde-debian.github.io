Install:

Add this sources.list.d entry (DEB822-style):

Types: deb
URIs: https://sonicde-debian.github.io/[system]/
Suites: main
Components: stable
Signed-By: /usr/share/keyrings/NexusSfan.pgp


Where [system] can be debian (Debian/Devuan)

If you are using Debian/Devuan Unstable then use Components: unstable
If you are using Debian/Devuan Testing then use Components: testing

Now, run curl -fsSL https://mrchicken.nexussfan.cz/publickey.asc | gpg --dearmor | sudo tee /usr/share/keyrings/NexusSfan.pgp > /dev/null
This will set up the publickey.

After updating the package cache, you can install SonicDE with sudo apt install sonicde sonic-archive-keyring
